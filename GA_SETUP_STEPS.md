# Google Analytics 添加 CasaCV 步骤

## 你现在看到的页面
Google Analytics 首页，显示你现有的账号：Mety Móveis

## 添加 CasaCV 步骤

### 第1步：点击左上角菜单
1. 点击页面左上角 **"所有账号"** 下拉框
2. 选择 **"创建账号"** 或 **"+ 创建媒体资源"**

### 第2步：创建新的媒体资源
1. 媒体资源名称：**CasaCV**
2. 时区：**佛得角 (GMT-1:00)**
3. 货币：**佛得角埃斯库多 (CVE)**

### 第3步：设置数据流
1. 选择 **"网站"**
2. 网站网址：**https://hi8150112.github.io/casacv/**
3. 数据流名称：**CasaCV Website**

### 第4步：获取跟踪代码
创建完成后，Google 会给你一串代码：
```
G-XXXXXXXXXX
```
（类似这样的格式）

### 第5步：把代码发给我
把 `G-XXXXXXXXXX` 复制给我，我帮你加到网站里。
